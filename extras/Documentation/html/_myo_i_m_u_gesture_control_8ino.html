Detailed Description

Example sketch using leds and the hardware serial connection to show the detected gestures.

Author
Valentin Roland (webmaster at vroland.de)
Date
September-October 2015 This library provides gesture detection functionality using almost exclusively the IMU data of the Myo Armband. The gestures are based on arm rotation to work with persons where distinct muscle activity is hard to detect. Muscle activity is only used for starting/ending the recording of a gesture. Uses the MyoBridge Arduino Library (https://github.com/vroland/MyoBridge).
